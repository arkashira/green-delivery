# Green Delivery Performance Monitor

This project provides a performance monitoring system for green delivery routes.

## Usage
1. Create a `PerformanceMonitor` instance.
2. Add `PerformanceMetric` instances to the monitor.
3. Generate a report using the `get_report` method.
4. Check for significant changes using the `alert_significant_changes` method.

## Testing
Run the tests using `pytest` in the `tests` directory.
